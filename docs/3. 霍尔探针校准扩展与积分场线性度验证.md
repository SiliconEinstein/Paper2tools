## 背景

霍尔探针（Hall Probe）是超导磁体磁场测量中最常用的传感器（cluster_60 中 80% 的 A-主线链使用）。测量系统通过以下信号链将传感器输出转换为磁场值：

```
霍尔传感器 → 放大器（×Gain） → 低通滤波器 → ADC → 数字记录
```

对于加速器偶极磁体，关键性能指标是**积分场线性度**（Integrated-field linearity）和**良场区均匀性**（Good-field region uniformity），它们决定了束流能否被精确操控。

本题采用 workflow 中提取的参数和公式构造数据，要求参赛者走通从原始电压到最终性能指标的完整链路。

**电压-磁场转换公式**：

```
B = V_measured / (G × S)
```

其中 `V_measured` 是 ADC 记录的放大后电压（单位 V），`G` = 1000（无量纲），`S` = 7.07 × 10⁻³ V/T。

## 科学路径

```
1  ADC电压 V_measured
2  → 电压-磁场转换 B = V/(G×S)
3  → 梯形法则积分 BL = ∫B(z)dz
4  → 线性拟合与非线性度评估
5  → 横向均匀性评估
6  → 与 FEM 仿真基准对比
7  → 误差预算汇编
```

## 验证标准

提交结果需要满足以下定量与定性要求：

1. **电压转换**：`B_peak` = 7.07 V / (1000 × 7.07×10⁻³ V/T) = 1.0 T（精确）
2. **积分场**：梯形法则计算的 `BL` 应在 345-355 T·mm 范围内
3. **有效磁长度**：`L_eff` = BL / B_peak 应接近 workflow 中的 377 mm（允许 ±5%）
4. **线性度**：3000A 处非线性度应 > 0%（因铁芯饱和），但具体数值取决于拟合策略
5. **均匀性**：±100mm 范围内均匀性应 < 2%（与 workflow 一致）
6. **误差预算**：系统误差 RSS < 0.2%，随机误差平均后 < 0.02 μT
7. **数据完整性**：所有要求的 CSV、JSON 和图像文件均已提交，且列名/键名与 milestone 要求一致
