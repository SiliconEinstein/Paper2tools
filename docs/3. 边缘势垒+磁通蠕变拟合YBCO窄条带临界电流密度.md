# 3. 边缘势垒 + 磁通蠕变拟合 YBCO 窄条带临界电流密度

> 模板版本：v2026.04.28 ｜ 难度：中等 ｜ 预计耗时：3–4 小时 ｜ 标签：超导、YBCO、薄膜、窄条带、边缘势垒、Tinkham 磁通蠕变、退配对极限

## 任务目标

本题基于今日 LKM 超导 workflow 中的 `cluster_12` 生成。目标是沿着其中一条闭环科学路径，复现窄条带 YBa₂Cu₃O₇₋δ (YBCO) 薄膜临界电流密度 Jc(w,T) 的提取与解释过程：

1. 从 reverse-derived 的 Jc(w) 基准点出发，拟合"边缘势垒近似"宽度依赖公式，分离体钉扎贡献 Jc^b 与边缘贡献 Jc^e；
2. 从 reverse-derived 的 Jc(T) 基准点出发，按 Tinkham 磁通蠕变形式拟合 α、β，并由 α 反推磁通蠕变激活能 U_p；
3. 由超导参数 λ、ξ、d 计算横向穿透深度 λ⊥ 与退配对极限 Jc^dp，并与拟合得到的 Jc^e 做"接近本征极限"的对比检验；
4. 在固定 reverse-derived 基准下扫描几何与材料参数（a、λ、ξ、d）的灵敏度，避免题目沦为单一公式代入。

参赛者需要根据题目给出的背景、公式、几何参数和反向推导基准数据，完成单位换算、公式实现、基准验证、参数灵敏度分析与数据整合，并提交可检查的结果。

## 背景

YBCO 薄膜被刻蚀成微米级窄条带后，临界电流密度 Jc 通常**随条带宽度 w 减小而增大**，至 w 接近横向穿透深度 λ⊥ 时出现明显增强；当 w 进一步减小到边缘损伤区（典型 ≲200 nm）时 Jc 反而下降。在 w ≳ λ⊥ 区域，可用以下"边缘势垒近似"公式描述：

```text
Jc(w) ≈ Jc^e · e^(-1/2) · (π/2) · √(a · λ⊥ / w) + Jc^b      ......(1)
```

其中：

- `Jc^e` 是边缘势垒/边缘钉扎决定的临界电流密度（通常接近退配对极限 Jc^dp）；
- `Jc^b` 是体钉扎贡献的临界电流密度；
- `a` 是无量纲拟合参数；
- `λ⊥ = λ² / d` 是薄膜的横向（Pearl）穿透深度。

温度依赖采用 Tinkham 磁通蠕变公式的小 t 展开（其中 t = T / Tc）：

```text
Jc(T) = Jc(0) · (1 - α·t - β·t²)                            ......(2)
α = (k_B · T_c / U_p) · ln(E0 / E)                          ......(3)
```

> **关于公式 (3) 与原文的偏差**：
> Tahara et al. 原文（para 5, P1）字面写法为 `α = (kT/Up) · ln(E₀/E)`，分子为 T。但原文同时声明 α 是温度无关的常数（"α ≈ 1"），因此分子上的"T"在物理一致性上必须吸收为 T_c（来自小 t 展开时 J_c(t) = J_c(0)·(1−αT/T_c−...) 的代换）。本题采用 T_c 形式以避免参赛者在 U_p 反推时遇到温度依赖。该改写不影响 Milestone 1、3、4、5 的任何中间量。

退配对极限（SI 单位，与 Tahara 论文取的形式等价）：

```text
Jc^dp = Φ₀ / (4π · μ₀ · λ² · ξ)                             ......(4)
```

本题围绕一篇 800 Å 厚 YBCO 薄膜（ZrO₂ 衬底）的窄条带工作展开。该论文使用 DC 四端法（临界判据 ≈ 3 μV/cm）测量了 1–13 μm 宽度、10 K 温度下的 Jc(w)，并在 w = 6 μm 处测量了 10–60 K 范围的 Jc(T)。**由于原文未以表格形式给出 Jc(w) 与 Jc(T) 的原始数据点，本题提供两组 reverse-derived benchmark：**

- Jc(w) benchmark 由"边缘势垒近似公式 + 一组自洽参数"反向生成，用于检验拟合实现；
- Jc(T) benchmark 由"Tinkham 公式 + 一组自洽参数"反向生成，用于检验温度依赖拟合实现。

它们**不是**独立的实验测量值，仅作为公式实现与单位换算的一致性基准。

### 4. 单位换算提示

- 1 A/cm² = 10⁴ A/m²；
- Φ₀ = 2.0678×10⁻¹⁵ Wb；μ₀ = 4π×10⁻⁷ H/m；k_B = 1.3807×10⁻²³ J/K = 8.617×10⁻⁵ eV/K；
- 在公式 (1) 中所有长度需统一为同一单位（建议全部用 μm 或全部用 m）；
- 在公式 (4) 中 λ、ξ 必须用 m，输出 Jc^dp 单位为 A/m²，最后再换算到 A/cm²；
- 本题不要求做去磁因子修正、各向异性张量分解或 c 轴投影换算。

## 科学路径

```text
窄条带 Jc(w,T) reverse-derived benchmark
  → 拟合 (1)：分离 Jc^e、Jc^b、a（固定 λ⊥ = 0.75 μm）
  → 拟合 (2)：由 Jc(T) 提取 Jc(0)、α、β
  → 由 α 通过 (3) 反推钉扎能 U_p（在 w=6 μm 处）
  → 由 λ、ξ、d 计算 λ⊥（验证）和 Jc^dp（公式 4）
  → 计算 Jc^e / Jc^dp 比值（结构检验：与几何参数无关）
  → 扫描 a、λ、ξ、d 的灵敏度
  → 整合 Jc(w,T)、可视化与插值
  → 评估 w < 2 μm 数据未纳入拟合的原因
```
