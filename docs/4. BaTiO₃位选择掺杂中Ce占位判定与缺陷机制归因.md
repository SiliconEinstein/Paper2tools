## 背景

ABO₃ 钙钛矿铁电陶瓷（如 BaTiO₃）的性能高度依赖掺杂离子的**占位**（A 位 vs B 位）。稀土 Ce 因其可变价态（Ce³⁺/Ce⁴⁺）尤为特殊：

- 当 Ba/Ti < 1 时，Ce 倾向以 Ce³⁺ 进入 Ba（A）位
- 当 Ba/Ti > 1 时，Ce 倾向以 Ce⁴⁺ 进入 Ti（B）位

占位不同导致截然不同的缺陷补偿机制和功能特性。本题采用的核心公式和判据：

**公式 (1) Goldschmidt 容忍因子**：

```
t = (r_A + r_O) / [√2 · (r_B + r_O)]
```

其中 r_A 为 A 位离子有效半径（12 配位），r_B 为 B 位离子有效半径（6 配位），r_O = 1.40 Å（O²⁻，6 配位）。钙钛矿稳定范围：0.88 < t < 1.06。

**公式 (2) 尺寸失配度**：

```
Δr_A = |r_dopant(12-coord) - r_host_A(12-coord)|
Δr_B = |r_dopant(6-coord) - r_host_B(6-coord)|
```

掺杂离子倾向进入 Δr 较小的位点。

**公式 (3) Arrhenius 电导率**：

```
σ = σ₀ · exp(-E_a / k_B T)
```

两边取对数：`ln σ = ln σ₀ - E_a / (k_B T)`，即 `ln σ` vs `1/T` 斜率 = `-E_a / k_B`。

其中 k_B = 8.617 × 10⁻⁵ eV/K。活化能 E_a 反映载流子类型：

| E_a 范围 | 载流子 |
|----------|--------|
| 0.3–0.5 eV | 单价氧空位电离 V_O → V_O• + e' |
| 0.6–1.2 eV | 双价氧空位电离 V_O• → V_O•• + 2e' |
| > 2 eV | 氧空位长程迁移 |

**公式 (4) 缺陷反应（Kröger-Vink 记号）**：

Ce³⁺ 进入 Ba 位（施主掺杂），A 位空位补偿：

```
3CeO₂ + 3TiO₂ → 2Ce_Ba• + 3Ti_Ti + 9O_O + V_Ba'' + CeO₂ + ½O₂     … (4a)
```

Ce⁴⁺ 进入 Ti 位（等价掺杂），无需电荷补偿：

```
CeO₂ → Ce_Ti + 2O_O                                                  … (4b)
```

**E 类诠释性说明**：原文公式 (1)–(3) 中还列出了 Ti 空位补偿路径 (4CeO₂ → ... V_Ti'''' ...) ，但原文 XPS 数据（Ti 2p 峰位不变）已排除 Ti 空位存在。本题简化为只考虑 Ba 空位补偿（公式 4a），与原文结论一致。
